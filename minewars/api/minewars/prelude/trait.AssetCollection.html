Expand description

Trait to mark a struct as a collection of assets

Derive is supported for structs with named fields.

ⓘ
#[derive(AssetCollection, Resource)]
struct MyAssets {
    #[asset(path = "player.png")]
    player: Handle<Image>,
    #[asset(path = "tree.png")]
    tree: Handle<Image>
}
