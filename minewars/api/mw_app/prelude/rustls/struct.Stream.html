pub struct Stream<'a, C, T>where
    C: 'a + ?Sized,
    T: 'a + Read + Write + ?Sized,{
    pub conn: &'a mut C,
    pub sock: &'a mut T,
}
Expand description

This type implements io::Read and io::Write, encapsulating a Connection C and an underlying transport T, such as a socket.

This allows you to use a rustls Connection like a normal stream.

Fields§

§conn: &'a mut C

Our TLS connection

§sock: &'a mut T

The underlying transport, like a socket
