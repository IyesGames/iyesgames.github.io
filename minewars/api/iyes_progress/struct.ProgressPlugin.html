Expand description

Add this plugin to your app, to use this crate for the specified state.

If you have multiple different states that need progress tracking, you can add the plugin for each one. Tracking multiple state types simultaneously is not currently supported (see issue #20).

If you want the optional assets tracking (“assets” cargo feature), enable it with .track_assets().

Warning: Progress tracking will only work after the StateTransition!

TrackedProgressSet represents all systems with progress tracking enabled. Calling track_progress will add your systems to the set automatically.

app.add_plugin(ProgressPlugin::new(MyState::GameLoading).continue_to(MyState::InGame));
app.add_plugin(ProgressPlugin::new(MyState::Splash).continue_to(MyState::MainMenu));
